The Filter\+Precondition mask is used to explicitate what kind of data a filter really needs to be applied. For example algorithms that compute per face quality have as precondition the existence of faces (but quality per face is not a precondition, because quality per face is created by these algorithms) on the other hand an algorithm that deletes faces according to the stored quality has both Face\+Quality and Face as precondition.
